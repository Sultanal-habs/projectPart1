Art Gallery API Client
This is a remote web-based application that interacts with the Art Gallery Web API. It demonstrates Search, Add, Update, and Delete operations via API endpoints.
Search Artworks (GET /api/ArtGalleryApi/artworks)
Add Artwork (POST /api/ArtGalleryApi/artworks)
Update/Delete Artwork
Search Artists (GET /api/ArtGalleryApi/artists)
Add Artist (POST /api/ArtGalleryApi/artists)
Update/Delete Artist
Search Exhibitions (GET /api/ArtGalleryApi/exhibitions)
